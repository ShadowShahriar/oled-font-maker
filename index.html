OLED Font Maker

Overview

This tool was inspired by the original OLED-js Font Foundry created by noopkat for oled-js. However, oled-js no longer works with the Raspberry Pi 5. So, I had to use migrate to one of its supported forks oled-rpi-i2c-bus

This new package doesn't provide a similar tool to create custom font modules, so I made my own.

Customization

Type in the full name of one of the fonts you've got installed on your system:
